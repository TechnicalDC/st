# st - My own st build

st is a simple terminal implementation for X.

## Patches applied

* Xresources
* w3m
* Scrollback (all 3 patches)
* Font2
* Desktop entry
* Clipboard
* Alpha
* Boxdraw
* Vertcenter

## Installation

```bash
git clone https://github.com/TechnicalDC/st
cd st
make
sudo make install
```
